The project is organized in a simple way so you can easily find everything you need:

data1.csv  :The dataset containing product reviews.
main.py : The main script where data preprocessing, training, evaluation, and visualization happen.
README.md:  Documentation that explains the project (this file).
saved_models :   A folder where all trained models and preprocessing objects are stored, such as:
simple_nn_model.pth : Trained SimpleNN model.
lstm_model.pth : Trained LSTM model.
bilstm_model.pth : Trained BiLSTM model.
vocab.pkl : Saved vocabulary (word-to-index mapping).
label_encoder.pkl : Saved label encoder for sentiments
